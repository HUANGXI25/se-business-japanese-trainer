# Maintainer Notes

## 当前维护重点

- 保持本地运行稳定
- 提升规则反馈的可信度和可解释性
- 持续改善场景真实性
- 让仓库更适合公开协作和长期维护

## 最欢迎的贡献方向

- 新的高质量 SE 场景
- 更贴近真实职场对象差异的推荐表达
- 能防回归的轻量测试
- 文案一致性、README 和文档改进

## 近期不打算做的事情

- 重写前端技术栈
- 把项目改成必须依赖外部 LLM 才能运行
- 引入复杂权限系统或在线多用户功能
- 为了“更工程化”而做大规模重构

## 后续版本优先级

1. 稳定核心训练链路和规则测试
2. 继续补真实场景和反馈质量
3. 改善场景贡献和规则维护体验
4. 在此基础上再评估可选的 LLM 增强

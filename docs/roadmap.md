# Roadmap

这个 roadmap 只描述当前真实可预期的方向，不承诺具体日期。

## v0.1.x

目标：把项目整理到适合公开展示和持续维护的状态。

- 完善 README、架构文档和维护文档
- 补齐最小测试集和 GitHub Actions
- 清理仓库卫生问题
- 保持本地运行体验稳定

## v0.2

目标：提升训练链路的可靠性和规则质量。

- 增强训练流程相关测试
- 提升评分解释性和片段级反馈质量
- 优化错题本和分析页的可用性
- 继续补强场景真实性和推荐表达质量

## v0.3

目标：让场景扩展和规则维护更顺手。

- 进一步规范场景数据结构
- 为新增场景提供更清晰的贡献模板
- 改善设置项和本地配置方式
- 适度减少路由层中的业务拼装逻辑

## Future

目标：在不破坏本地优先特性的前提下，为更强反馈能力预留升级空间。

- 可选的 LLM 增强接口
- 更细的薄弱点分析
- 更丰富的场景分类和练习方式
- 如果确有需求，再评估更完整的部署方案

## Non-Goals for Now

近期不把以下内容作为优先项：

- 重写为大型前端框架
- 强依赖外部 API 才能运行
- 复杂的多用户系统
- 过早做成 SaaS 化产品

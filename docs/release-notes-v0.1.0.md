# v0.1.0 Release Notes Draft

## Summary

`v0.1.0` is the first public-ready baseline for Business Japanese SE Roleplay Trainer.

This release turns the project from a local prototype into a repository that is ready to be shared, reviewed, and iterated on in public.

## What is included

- A local-first FastAPI web app for practicing Japanese business communication in SE workplace scenarios
- 21 built-in roleplay scenarios
- Learning mode and practice mode
- Rules-based scoring and fragment-level feedback
- Mistake notes, practice records, and weakness analysis
- Minimal automated tests and a GitHub Actions workflow
- Core OSS documentation for contributors and maintainers

## What this project is today

- A local MVP
- Explainable and rules-driven
- Suitable for scenario expansion and feedback iteration
- Ready for open collaboration, but not yet a production platform

## Current limitations

- Feedback quality is still limited by hand-written rules
- Test coverage is intentionally minimal
- No authentication, multi-user support, or deployment hardening
- LLM integration is only a future extension point

## Good first contribution areas

- Improve realism of existing scenarios
- Add new SE communication scenarios
- Refine scoring and correction rules
- Add a few more stable API and workflow tests
- Improve docs and onboarding for contributors
